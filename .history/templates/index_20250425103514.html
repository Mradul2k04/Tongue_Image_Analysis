Tongue Health Analyzer

AI-based health screening through tongue analysis

Analyze

In this project, I created a web-based health screening application that utilizes Convolutional Neural Networks (CNNs) to examine tongue images and offer recommendations about a user's health status.

🔍 Project Overview

The color of the tongue may be a strong predictor of a range of health issues — an idea rooted in traditional and contemporary medicine. I developed an AI system that accepts three images of a user's tongue (front, left, and right) and applies a trained CNN to classify or analyze potential health indicators.

🛠️ Technologies Used

  • Python
  • TensorFlow / Keras (for training and constructing the CNN)
  • Flask (for web backend development)
  • HTML/CSS & Bootstrap (for frontend UI)
  • JavaScript (to manage camera and image uploads)

📦 Model Details

The CNN was trained on a custom dataset of labeled tongue images. The network consists of multiple convolutional and pooling layers, followed by dense layers, and is optimized using categorical crossentropy loss and the Adam optimizer. The trained model was saved as a .h5 file and integrated with the Flask backend for real-time browser predictions.

🌐 Web App Features

  • Users can upload or directly capture three tongue images (front, left, right)
  • The model processes the images and provides health analysis feedback
  • Responsive and clean UI with mobile camera support
  • Dynamic parallax background and smooth scrolling experience

🚀 Impact

This project shows the creative and practical potential of deep learning in healthcare. It sets the stage for contactless, intelligent medical diagnostics powered by computer vision.

🔮 Future Improvements
